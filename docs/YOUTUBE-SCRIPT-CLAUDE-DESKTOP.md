## Notes for recording

- **Pace:** speak normally, not slow. Info per second.
- **Don't read every word.** Hit the beats; improvise.
- **Cut aggressively.** If a step takes 3 seconds, show 3 seconds.
- **Background music:** low and chill, no vocals.
- **Show the actual numbers.** Real positions, real prices, real signatures.
  Authenticity is the value.
- **The safety beat is the load-bearing moment.** Slow down on "what this
  agent CAN'T do" — that's what makes a non-technical viewer trust it.

## Key visual moments to land

| Moment | Why it matters |
|---|---|
| Real BTC position appearing in chat | Proves it's real, not faked |
| The setup page with the safety panel | Establishes trust |
| MetaMask popup × 2 | Familiar UX — viewers know this means "safe" |
| Success message back in Claude | Closure |
| Real order placed via chat in step 3 | Shows the daily-use experience |
