Hyperliquid CLI Setup

One-time authorization. Sign safely with your wallet.

No private keys are ever shared.

This page is the second step of setup, not the first.

It needs to be opened from your CLI — the CLI generates a personal link for you with the agent address baked in. Without that, there's nothing to sign.

If you're using the terminal

Run this command. Replace the placeholder with your main wallet's public address (the 0x… one — it's safe to share, it's just like an email).

hyperliquid agent setup --user 0xYOUR_MAIN_ADDRESS

It'll print a link and auto-open this page with the right info filled in.

If you're using Claude Desktop

Open a new chat and tell Claude:

"Set up Hyperliquid for me. My main wallet is 0xYOUR_MAIN_ADDRESS."

Claude runs the setup tool and gives you a link to click.

Don't have the CLI yet? Get it on GitHub — install instructions in the README.

Once your wallet is connected, you'll sign two messages — one to authorize your trading agent, one for the builder fee. Both are free and never move funds.

What we're doing

You're about to give a "trading agent" permission to place and cancel orders on your Hyperliquid account. That agent has its own key on your computer — the one your CLI just generated.

You'll sign two messages with your wallet (MetaMask, Rabby, etc.). They don't move money. They just register on-chain that you've approved this setup.

Exactly what you're approving

Trading agent address (the helper that does the trading)
Builder address (receives the per-trade fee)
Maximum fee rate (per trade, capped server-side)
Network
⚠ Verify the agent address. The address shown above should exactly match what your CLI just printed. If it doesn't, close this tab and check that you opened the link from your own terminal — not from a message someone sent you.

Safety

✓ This trading agent CAN

  • Place, modify, cancel orders
  • Charge a small fee on each order (capped at the rate above)

✗ This trading agent CANNOT

  • Withdraw your USDC
  • Transfer funds to anyone else
  • Approve more agents
  • Change the fee rate

Hyperliquid enforces those limits on their server. Even if your computer is compromised, the worst an attacker could do is submit bad orders. They cannot drain your account.
